## Tools Installed

The following core tools and extensions were successfully integrated into the development workspace:

1. **Cursor IDE**: An AI-first code editor fork of VS Code, optimized for pair-programming with large language models.
2. **Claude Cursor for VS Code**: An extension that integrates Anthropic's Claude models into the editor to assist with contextual code generation, explanations, and logical reasoning.

## Issues Encountered & Solutions

### 1. Hidden Extensions Menu & Default AI Chat View
- **Issue**: Upon launching Cursor IDE for the first time, the interface defaulted entirely to an interactive "Cursor Agent" / AI onboarding screen. The standard primary sidebar and traditional VS Code menus—including the Extensions Marketplace—were not visible, creating confusion on how to install add-ons.
- **Solution**: Initiated a fresh development workspace by explicitly triggering an **"Open Editor Window"** action. This initialized the standard IDE workbench layout, restoring the top menu bar and sidebar navigation where extensions could be managed.
